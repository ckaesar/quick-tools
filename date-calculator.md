# 日期计算器 (Date Calculator)

这是一个功能强大的日期计算工具，支持计算两个日期之间的间隔天数，以及根据指定天数、周数或月数推算未来的或过去的日期，并能显示对应的农历日期。

## 功能特点

1.  **日期差计算**：
    *   输入起始日期和结束日期（默认为今天）。
    *   计算两个日期相差的天数，并估算相差的周数。
    *   如果起始日期晚于结束日期，结果显示为负数。
2.  **日期推算**：
    *   输入基准日期（默认为今天）。
    *   选择推算方向（往后/往前）。
    *   输入推算的数值和单位（天、周、月）。
    *   计算结果包含阳历日期、星期几以及对应的农历日期。
3.  **自动纠错**：
    *   输入的月、日如果超出正常范围（如11月32日），会自动调整为正确的日期（12月2日）。
4.  **农历支持**：
    *   内置农历转换功能（依赖 CDN），方便查看传统日期。

## 使用方法

1.  直接在浏览器中打开 `date-calculator.html` 文件。
2.  **计算日期差**：
    *   在第一部分输入两个日期。
    *   点击“相差”按钮查看结果。
3.  **推算日期**：
    *   在第二部分输入基准日期。
    *   选择方向（往后/往前）、输入数量、选择单位（日/周/月）。
    *   点击“是”按钮查看结果。
