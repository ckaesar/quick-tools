# JSON 格式化工具 (JSON Formatter)

这是一个轻量级的、单文件的 JSON 格式化工具。它无需安装任何依赖，也没有复杂的构建过程，直接打开 HTML 文件即可使用。

## 功能特点

1.  **JSON 格式化**：将压缩或混乱的 JSON 字符串格式化为易读的结构（缩进 4 个空格）。
2.  **错误检查**：在格式化之前会验证 JSON 的合法性。如果输入的字符串不是有效的 JSON，会显示具体的错误信息。
3.  **单文件设计**：HTML、CSS 和 JavaScript 全部内联在一个文件中，方便携带和离线使用。
4.  **清空功能**：提供一键清空输入和输出区域的按钮，方便快速开始新的操作。
5.  **响应式布局**：简洁的界面，适应不同屏幕尺寸。

## 使用方法

1.  直接在浏览器中打开 `json-formatter.html` 文件。
2.  在顶部的文本框中粘贴需要格式化的原始 JSON 字符串。
3.  点击中间的 **“格式化 JSON”** 按钮。
    *   如果 JSON 有效，格式化后的结果将显示在下方的文本框中。
    *   如果 JSON 无效，中间区域会显示红色的错误提示。
4.  点击 **“清空”** 按钮可以清除所有内容。

## 原始需求 / 提示词

该工具是根据以下需求创建的：

> 创建一个json格式化工具，上面一个输入框用来输入json字符串，中间一个格式化按钮，点击按钮之后将格式化之后的json到下面的一个输入框， 格式化的时候需要检查原始的字符串是否是标准的json公式，如果有错误则提示出来。 
> 将内联 JavaScript 和 CSS 放在单个 HTML 文件中，不适用复杂的js框架如react
