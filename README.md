# Quick Tools

一个包含各种实用前端小工具的集合。这些工具通常是单文件的 HTML，方便携带和离线使用。

## 工具列表

| 工具名称 | 文件 | 在线使用 | 说明文档 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| **JSON 格式化工具** | [json-formatter.html](./json-formatter.html) | [在线使用](https://ckaesar.github.io/quick-tools/json-formatter.html) | [README](./json-formatter.md) | 一个简单的 JSON 格式化与校验工具，支持错误提示。 |
| **日期计算器** | [date-calculator.html](./date-calculator.html) | [在线使用](https://ckaesar.github.io/quick-tools/date-calculator.html) | [README](./date-calculator.md) | 一个日期差计算与日期推算工具，支持农历显示。 |
| **生存计算器** | [survival-calculator.html](./survival-calculator.html) | [在线使用](https://ckaesar.github.io/quick-tools/survival-calculator.html) | [README](./survival-calculator.md) | 一个基于本金、开销和收益率的财务生存年限计算工具。 |

## 使用说明

直接点击上方表格中的“文件”链接即可在浏览器中打开对应工具。
如需了解具体功能和原始需求，请查看对应的说明文档。
